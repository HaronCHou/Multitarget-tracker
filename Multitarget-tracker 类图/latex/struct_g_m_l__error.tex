\doxysubsection{详细描述}
Reason and position of an error in a GML file. 

When an error occurs while parsing the structure of a GML file {\ttfamily \mbox{\hyperlink{struct_g_m_l__error}{GML\+\_\+error}}} is used to return the type and position of the error detected. Position is specified by {\ttfamily line} and {\ttfamily column}, but might be somewhat imprecise. However at least the line number should not differ too much from the real position.

\doxysubsection{详细描述}
Topological sorting. 

Assigns to each node {\ttfamily n} a number {\ttfamily top\+\_\+num} such that for every edge {\ttfamily (u,v)} {\ttfamily top\+\_\+num\mbox{[}u\mbox{]}} \texorpdfstring{$<$}{<} {\ttfamily top\+\_\+num\mbox{[}v\mbox{]}}, if possible, i.\+e. iff the directed graph is acyclic. ~\newline


Similar to the testing of biconnectivity, which extends DFS to calculate low-\/numbers, the topsort-\/algorithm extends DFS to calculate the new numbering (and thus to test whether such a numbering is possible).

In order to traverse all the nodes in the order of its top-\/numbers, a new iterator, {\ttfamily topsort\+\_\+iterator} is provided.

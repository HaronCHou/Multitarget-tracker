\doxysubsection{详细描述}
Local Binary Similarity Pattern (\mbox{\hyperlink{class_l_b_s_p}{LBSP}})-\/based change detection algorithm (abstract version/base class).

For more details on the different parameters, see P.-\/L. St-\/\+Charles and G.-\/A. Bilodeau, \char`\"{}\+Improving Background \+Subtraction using Local Binary Similarity Patterns\char`\"{}, in WACV 2014, or G.-\/A. Bilodeau et al, \char`\"{}\+Change Detection in Feature Space Using Local Binary Similarity Patterns\char`\"{}, in CRV 2013.

This algorithm is currently NOT thread-\/safe.

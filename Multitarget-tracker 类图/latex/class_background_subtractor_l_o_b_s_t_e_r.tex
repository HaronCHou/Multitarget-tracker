\doxysubsection{详细描述}
LOcal Binary Similarity segmen\+TER (LOBSTER) change detection algorithm.

Note\+: both grayscale and RGB/\+BGR images may be used with this extractor (parameters are adjusted automatically). For optimal grayscale results, use CV\+\_\+8\+UC1 frames instead of CV\+\_\+8\+UC3.

For more details on the different parameters or on the algorithm itself, see P.-\/L. St-\/\+Charles and G.-\/A. Bilodeau, \char`\"{}\+Improving Background Subtraction using Local Binary Similarity Patterns\char`\"{}, in WACV 2014.

This algorithm is currently NOT thread-\/safe.

\doxysubsection{详细描述}
Opaque handle that holds logging information for a particular test 

This object is an opaque handle to information used by the \mbox{\hyperlink{class_logger}{Logger}} to print test results. The sample must call \mbox{\hyperlink{class_logger_afb592500347528aa18527c9909562a96}{Logger\+::define\+Test()}} in order to obtain a \mbox{\hyperlink{class_logger_1_1_test_atom}{Test\+Atom}} that can be used with \mbox{\hyperlink{class_logger_aed19e792e4f52622b3db6d5a8b629322}{Logger\+::report\+Test}}\{Start,End\}().

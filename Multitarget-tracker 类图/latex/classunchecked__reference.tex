\subsubsection*{template$<$typename T, \mbox{\hyperlink{detail_2common_8h_ac430d16fc097b3bf0a7469cfd09decda}{ssize\+\_\+t}} Dims$>$\newline
class unchecked\+\_\+reference$<$ T, Dims $>$}
Proxy class providing unsafe, unchecked const access to array data. This is constructed through the {\ttfamily unchecked\texorpdfstring{$<$}{<}T, N\texorpdfstring{$>$}{>}()} method of {\ttfamily array} or the {\ttfamily unchecked\texorpdfstring{$<$}{<}N\texorpdfstring{$>$}{>}()} method of {\ttfamily \mbox{\hyperlink{classarray__t}{array\+\_\+t}}\texorpdfstring{$<$}{<}T\texorpdfstring{$>$}{>}}. {\ttfamily Dims} will be -\/1 for dimensions determined at runtime.

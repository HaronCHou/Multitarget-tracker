\doxysubsection{详细描述}
\textbackslash{}rst Wraps a Python iterator so that it can also be used as a C++ input iterator

Caveat\+: copying an iterator does not (and cannot) clone the internal state of the Python iterable. This also applies to the post-\/increment operator. This iterator should only be used to retrieve the current value using {\ttfamily \mbox{\hyperlink{classiterator_ac92dc18748448d5e9878c2df3c36eca2}{operator$\ast$()}}}. \textbackslash{}endrst

Convenience object used to facilitate use of C++ stream syntax when logging messages. Order of base classes is \mbox{\hyperlink{class_log_stream_consumer_base}{Log\+Stream\+Consumer\+Base}} and then std\+::ostream. This is because the \mbox{\hyperlink{class_log_stream_consumer_base}{Log\+Stream\+Consumer\+Base}} class is used to initialize the \mbox{\hyperlink{class_log_stream_consumer_buffer}{Log\+Stream\+Consumer\+Buffer}} member field in \mbox{\hyperlink{class_log_stream_consumer}{Log\+Stream\+Consumer}} and then the address of the buffer is passed to std\+::ostream. This is necessary to prevent the address of an uninitialized buffer from being passed to std\+::ostream. Please do not change the order of the parent classes.

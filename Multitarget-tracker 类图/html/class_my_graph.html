◆ read_labels_as_weights()

void MyGraph::read_labels_as_weights ( )
inline

Sets the labels_as_weights flag to true (the default is false). If this flag is set the load_edge_info_handler will read any labels associated with an edge in a GML file an integer weight. This is useful if you want to import a GML graph generated by LEDA.

Handles a string value associated with an edge. By default, it process the "label" key by storing value in the label edge map. If labels_as_weights is true</true>, then converts label to integer and sets the edge weight to that value. This method is called by load_edge_info_handler.

ethe edge
keythe name of the item (e.g., "label")
